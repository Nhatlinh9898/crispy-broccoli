# AI Agent Training Curriculum - Hệ thống Tạo Bài viết Tự động

## 🎯 Tổng quan Chương trình Huấn luyện

Chương trình huấn luyện này được thiết kế để đào tạo AI agents trở thành chuyên gia trong việc sử dụng hệ thống tạo bài viết tự động cho linh kiện công nghiệp. Chương trình kéo dài 12 tuần, chia thành 4 giai đoạn với các bài kiểm tra và dự án thực tế.

## 📅 Lộ trình Huấn luyện

### Giai đoạn 1: Foundation (Tuần 1-3)
- **Tuần 1:** System Architecture & Basic Templates
- **Tuần 2:** Content Generation Fundamentals
- **Tuần 3:** Practical Implementation

### Giai độ 2: Advanced Templates (Tuần 4-6)
- **Tuần 4:** Advanced Template Development
- **Tuần 5:** Interactive Elements & SEO
- **Tuần 6:** System Integration

### Giai độ 3: Specialization (Tuần 7-9)
- **Tuần 7:** Industry-Specific Templates
- **Tuần 8:** Performance Optimization
- **Tuần 9:** Custom Development

### Giai độ 4: Mastery (Tuần 10-12)
- **Tuần 10:** Advanced Integration
- **Tuần 11:** Training & Documentation
- **Tuần 12:** Capstone Project & Certification

---

## 📅 Giai đoạn 1: Foundation (Tuần 1-3)

### Tuần 1: System Architecture & Basic Templates

#### **Mục tiêu tuần:**
- Hiểu kiến trúc hệ thống
- Làm chủ 4 templates cơ bản
- Sử dụng các công cụ cơ bản
- Thiết lập môi trường phát triển

#### **Nội dung chi tiết:**

##### **Ngày 1-2: System Overview**
- **Teoria (2 giờ):**
  - Architecture overview
  - Component relationships
  - Data flow analysis
  - File structure explanation
- **Lab (3 giờ):**
  - Setup development environment
  - Clone and run demo interface
  - Explore file structure
  - Test basic functionality
- **Assignment (1 giờ):**
  - Vẽ sơ đồ hệ thống
  - Giải thích vai trò mỗi component

##### **Ngày 3-5: Basic Templates**
- **Teoria (3 giờ):**
  - Template structure deep dive
  - Variables and dynamic content
  - Content types and formatters
  - Section organization principles
- **Lab (4 giờ):**
  - Analyze 4 basic templates
  - Extract variables from products
  - Create simple modifications
  - Test with sample products
- **Assignment (1 giờ):**
  - Phân tích template bolt
  - Tạo modification cho template resistor
  - Document changes

##### **Ngày 6-7: Content Generation**
- **Teoria (2 giờ):**
  - Article generation process
  - Variable replacement algorithms
  - Content structuring principles
  - Quality control measures
- **Lab (3 giờ):**
  - Generate articles for 10 products
  - Review and improve content
  - Implement basic quality checks
  - Export to different formats
- **Assignment (2 giờ):**
  - Tạo 5 bài viết chất lượng cao
  - Peer review 2 bài viết của người khác
  - Implement feedback

##### **Week 1 Assessment:**
- **Written Test (30 phút):** 20 câu hỏi trắc nghiệm
- **Practical Test (1.5 giờ):** Tạo 3 bài viết mới
- **Code Review (30 phút):** Review code modifications
- **Weekly Quiz (30 phút):** Kiểm tra kiến thức

## 📊 Assessment Methods

### Written Assessments

#### **Format:**
- **Multiple Choice:** 20-40 câu hỏi
- **Short Answer:** 5-10 câu hỏi ngắn
- **Essay Questions:** 2-4 câu hỏi dài
- **Case Studies:** 1-2 kịch bản tình huống

#### **Content Coverage:**
- **Technical Knowledge:** 40%
- **Practical Skills:** 30%
- **Problem Solving:** 20%
- **Best Practices**: 10%

#### **Scoring:**
- **Accuracy:** Đáp án đúng
- **Completeness:** Đủ thông tin
- **Clarity:** Rõ ràng, dễ hiểu
- **Depth:** Sâu sắc, chuyên môn

### Practical Assessments

#### **Projects:**
- **Mini Projects:** 1-2 tuần
- **Integration Projects:** 2-3 tuần
- **Capstone Project:** 4-6 tuần
- **Custom Projects:** 2-4 tuần

#### **Code Reviews:**
- **Peer Reviews:** Đánh giá bởi đồng nghiệp
- **Expert Reviews:** Đánh giá bởi chuyên gia
- **Self-Reviews:** Tự đánh giá

#### **Presentations:**
- **Weekly Progress:** 15-30 phút
- **Milestone Reviews:** 30-60 phút
- **Final Demo:** 1-2 giờ
- **Knowledge Sharing:** 30-60 phút

### Certification Criteria

#### **Level Requirements:**
- **Level 1:** 80%+ scores on all assessments
- **Level 2:** 85%+ scores on practical assessments
- **Level 3:** 90%+ scores on all assessments
- **Level 4:** 95%+ scores on capstone project

#### **Certification Process:**
1. **Application:** Nộp đơn và phỏng vấn
2. **Written Assessment:** Online test (60-90 phút)
3. **Project Submission:** Nộp dự án hoàn chỉnh
4. **Code Review:** Review bởi chuyên gia
5. **Interview:** Phỏng vấn trực tiếp
6. **Certification Issued:** Nếu đạt yêu cầu
